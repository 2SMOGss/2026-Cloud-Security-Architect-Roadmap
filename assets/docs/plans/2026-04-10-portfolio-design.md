# Design Doc: GitHub Profile Portfolio Refactor (The Cloud Sentinel)

**Date:** 2026-04-10
**Status:** Draft
**Goal:** Create a premium, truthful, and high-signal portfolio for Rob Chich, highlighting his transition from Paramedic to Cloud Security Architect.

## 🏛️ Architecture
The portfolio will be implemented using a GitHub Profile README (`README.md` in the `2SMOGss/2SMOGss` repository). It uses a combination of:
- **Markdown & Semantic HTML:** For structure and accessibility.
- **SVG Badges (Shields.io & Simple Icons):** For professional skill representation.
- **Dynamic Stats (github-readme-stats):** To demonstrate active technical engagement.
- **Generated Assets:** Custom dark-mode banner for visual excellence.

## 📋 Sections

### 1. Hero & Identity
- **Visual:** Custom dark-mode banner with "Sentinel" blue accents.
- **Headline:** `Rob Chich | Cloud Security & GRC Architect`
- **Narrative:** Bridging 20 years of operational resilience to secure cloud architectures.
- **Status:** Badges for "Current Focus: AWS SAA-C03" and "Completed: Google Cybersecurity Cert".

### 2. Tactical Metrics
- **GitHub Stats:** `tokyonight` theme cards for Stats and Top Languages.
- **Logic:** Demonstrates proof of hands-on work with PowerShell, Bash, and Python.

### 3. Credential Vault & Skill Arsenal (Truth-First)
- **Certifications:** Verified row for completed credentials (Google, AI Fundamentals).
- **Skill Matrix:** Categorized grid of tools (AWS, Splunk, NIST Frameworks).

### 4. Mission Logs (Projects)
- **Table/List:** Clean links to active repositories with short domain-focused descriptions (e.g., [Cloud-Honeynet-SOC] - Log Analysis).

### 5. Outreach
- **Socials:** High-impact badges for LinkedIn, YouTube, and X.

## ✅ Verification & Accuracy
- No fabricated certifications.
- Links must point to verified GitHub repositories.
- Paramedic background is used as a narrative strength (GRC/Resilience).
